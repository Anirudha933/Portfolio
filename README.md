# Anirudha Mohanty — Creative Developer Portfolio

Welcome to the source code of my personal portfolio website. This project is a high-performance, cinematic "scrollytelling" experience built to showcase my background as a **Systems-focused Software Developer** specializing in **High-Performance Systems** and **Intelligent ML & Backends**.

## 🚀 The Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Rendering:** HTML5 Canvas API

## ✨ Architecture & Features

- **Cinematic Scrollytelling:** Utilizes a custom-built, hardware-accelerated `<canvas>` engine to scrub through a 77-frame image sequence that is perfectly mapped to the user's scroll progress via Framer Motion. 
- **Zero-Flicker Preloading:** Image sequences are preloaded asynchronously into memory before the scroll sequence is allowed to render, ensuring flawless, butter-smooth transitions.
- **Glassmorphic Parallax Overlays:** Features premium, Awwwards-level glassmorphic UI cards and multi-color gradient typography. Text sections move at variable parallax speeds to create a deeply immersive 3D feel.
- **Responsive Canvas Scaling:** Leverages `object-top` and `object-cover` styling on the canvas to ensure the subject remains the focal point across ultra-wide desktop monitors and narrow mobile devices.
- **Seamless Cinematic Wipes:** Employs precise scroll-value transformations to fade the canvas smoothly into the background, providing an elegant transition into the standard DOM layout.

## 🛠️ Local Development

To run this project locally on your machine:

1. **Clone the repository**
   ```bash
   git clone https://github.com/Anirudha933/<your-repo-name>.git
   cd <your-repo-name>
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **View the site**
   Open [http://localhost:3000](http://localhost:3000) in your browser.
